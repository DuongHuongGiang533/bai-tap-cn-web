Trang Debug Lỗi Code

Mã gốc bị lỗi từ đề bài: 
function updateClock() {
    let now = new Date();
    let timeStr = now.getHours + ":" + now.getMinutes + ":" + now.getSeconds;
    document.getElementById("clock").innerHTML = timeStr;
}
setInterval(updateClock(), 1000);
ĐỒNG HỒ SAU KHI SỬA LỖI:
00:00:00
Giải thích các lỗi sai đã tìm thấy:
  • Lỗi 1: Các hàm now.getHours, now.getMinutes, now.getSeconds thiếu cặp dấu ngoặc tròn (). Vì đây là các phương thức của đối tượng Date, phải thêm () thì mới lấy được giá trị thời gian.
  • Lỗi 2: Trong hàm setInterval(updateClock(), 1000), việc truyền vào updateClock() có dấu ngoặc sẽ làm hàm chạy ngay lập tức một lần duy nhất rồi dừng lại. Cách đúng là bỏ dấu ngoặc đi, chỉ truyền tên hàm updateClock để nó tự lặp lại sau mỗi 1 giây.
